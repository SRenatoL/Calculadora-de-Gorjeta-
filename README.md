## Acesso ao Projeto

```

#Clone esse repositório
$ git clone git@github.com:SRenatoL/Calculadora-de-Gorjeta-.git

#Entrando na pasta do arquivo
$ cd Calculadora-de-Gorjeta-

#inicie o npm
$ npm init 

#instale o webpack
$ npx webpack-cli

$ npx webpack --mode development

#Inicie o Servidor

$ npm start

# - acesse <http://localhost:8080/>

```
